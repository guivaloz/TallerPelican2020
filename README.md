## Instale un entorno virtual

    $ cd ~/Escritorio
    $ mkdir Pelican
    $ cd Pelican
    $ python -m venv venv

Active el entorno virtual

    $ . venv/bin/activate

E instale los paquetes Pelican y Markdown...

    (venv) $ pip install pelican markdown

Revise los paquetes instalados con...

    (venv) $ pip list
